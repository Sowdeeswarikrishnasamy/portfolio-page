## ✨ Features

- 🎨 Luxury dark aesthetic — gold & black editorial design
- 🖱️ Custom cursor with smooth ring animation
- ⚡ Loading screen with name animation
- 📜 Scroll reveal animations on all sections
- 📊 Animated skill progress bars
- 🧭 Sticky navbar with scroll-based styling
- 📬 Contact form (front-end demo)
- 📱 Fully responsive for mobile/tablet

## 📝 Sections

- **Hero** — intro, role, quick stats
- **About** — education, internship, training background
- **Skills** — MERN stack, deployment tools
- **Projects** — HoloSense, ChatGram, CityGuard AI, Mini CRM, IoT Classroom Automation, AI Complaint Management System
- **Experience** — Codec Technologies internship, BSNL × Intel Bharat training
- **Contact** — email, LinkedIn, GitHub

## 🔧 Customize

Edit `index.html` to:
- Update email, LinkedIn, GitHub links in the Contact section
- Add a real photo in `.photo-frame`
- Add project live-demo / repo links

## 🌐 Deploy

Works out of the box on:
- **GitHub Pages** — enable in repo Settings → Pages
- **Vercel** / **Netlify** — drag and drop or connect repo
